How JavaScript Works?

there has few basic steps:

  • Javascript File: Where we write our raw code .
  • Javascript Engine:
    1. Parser: Parse the Syntax and find error. if get any error then stop execution and throw error
    2. Syntax Tree: in this stage the Syntax is converted in to an tree like operations are in a node assignment are in a different node etc.
    3. Compiled into Machine Code: the syntax converted into a pure machine readable code.
    4. Machine Code: machine run the code.
